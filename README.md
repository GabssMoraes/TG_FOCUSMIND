# 🧠 FocusMind

**FocusMind** é uma plataforma inovadora orientada a otimizar a rotina de estudos, ajudando estudantes a manterem o foco, organizarem suas matérias e vencerem a procrastinação utilizando ferramentas de produtividade e Inteligência Artificial.

## 🚀 Funcionalidades

- **Dashboard Integrado**: Acompanhe seu progresso diário, veja suas metas e mantenha sua sequência de dias de estudo ativo (streak).
- **Timer Pomodoro**: Ferramenta de foco com tempos customizáveis (ex: 25/5 min, 50/10 min) para você estudar em blocos e descansar na medida certa.
- **Assistente IA (FocusAI)**: Converse com a inteligência artificial para criar rotinas, tirar dúvidas rápidas e receber sugestões do que estudar no seu tempo livre.
- **Perfil e Preferências**: Cadastre suas matérias (Cálculo, Algoritmos, Física, etc.) e ajuste preferências personalizadas do seu ambiente de estudos.

## 💻 Tecnologias Utilizadas

- **React.js**: Biblioteca principal para a construção escalável da interface de usuário.
- **CSS Modules**: Estilização componentizada para garantir que o CSS seja isolado por página, facilitando a manutenção e organização do projeto.
- **React Router Dom**: Gerenciamento de rotas e navegação fluida entre as páginas (Dashboard, Timer, Chat, Perfil).

## 📥 Como executar o projeto localmente

1. Clone este repositório:
```bash
git clone https://github.com/GabssMoraes/TG_FOCUSMIND.git
```

2. Acesse a pasta do projeto React:
```bash
cd TG_FOCUSMIND/FOCUSMIND
```

3. Instale as dependências padrão:
```bash
npm install
```

4. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```
